Prérequis

Pour ce tutoriel, nous partons du principe que vous connaissez le HTML, le CSS et bien-sûr le Javascript. Si vous avez déjà jeté un coup d'oeil à Angular2, vous avez pu remarquer l'utilisation exclusif de TypeScript et non pas de Javascript pur. C'est pourquoi, les morceaux de code suivants sont redigés en TypeScript. Si vous ne connaissez pas ce sur-ensemble, faites un tour sur le site officiel Avant de débuter le tutoriel, installer node.js ainsi que npm sur votre ordinateur.

React Js

Installation

Commençons par installer ReactJs.
Ouvrez une invite de commande et tapez-y la commande suivante :

npm install -g create-react-app Cette ligne de commande nous permet de ne pas commencer une application "nue". Toutes vos futures applications ReactJs seront déjà configurées avec Babel ou encore WebPack.
Création d'une application

Retournez dans votre invite de commande et tapez la commande suivante :

create-react-app nomAppli Remplacez nomAppli pour votre propre nom d'appli.
Placez vous dans le dossier de votre appli : cd nomAppli

La state


class Button extends React.Component {
state = {
clicked: false
}

constructor() {
super();
this.handleClick = this.handleClick.bind(this);
}

handleClick() {
this.setState({ clicked: !this.state.clicked });
}

render() {
return (
  <div
    className="myButton"
    style={{ backgroundColor: this.state.clicked ? 'red' : 'blue' }}
    onClick={this.handleClick}
  >
    {this.props.text}
  </div>
);
}
}

ReactDOM.render(<Button text="Changement" />, document.getElementById('buttonReactState'));
